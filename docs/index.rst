*****************************************
mfs : Magnetic field simulation in Python
*****************************************

mfs is a Python package for calculating and visualising the magnetic field arising from various magnetic sources. It was originally written to support the ytterbium quantum optics experiment at the University of Southern Denmark in 2018.
